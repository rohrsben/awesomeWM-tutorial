Wibox

Wibox is awesome's library for creating widgets. Widgets are things like a clock or volume control, and widgets live in the taskbar. Wibox includes the ability to make most things you can think of, though some are harder than others. Now, let's add it to our rc.lua.test: 

                local beautiful = ...

-- Widgets library
local wibox = require("wibox")

beautiful.init(...

Menubar

The menubar is more like a fancy wibox than its own library. It implements the freedesktop menu stuff, and acts as a launcher for .desktop files. 

                local wibox = ...

-- Menubar
local menubar = require("menubar")

beautiful.init(...

Awful.autofocus

Also not so much a library, but a useful thing to have. requireing it automatically enables its functionality: making sure a window has focus if it should. 

                local menubar = ...

-- Autofocus windows
require("awful.autofocus")

Naughty

I left naughty for last, because it has to do with what we'll be adding next. Naughty is a desktop notification library/system, which allows you to send your own notifications and handles them from other apps, like a spotify song change.
